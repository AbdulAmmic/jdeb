jdeb by tcurdt

This library provides an Ant task and a Maven plugin to create Debian packages from Java builds in a truly cross platform manner. Build your Debian packages on any platform that has Java support. Windows, Linux - it does not require additional native tools installed.
The API underneath is well abstracted can easily be adopted for other areas as well.

Documentation

Check the documentation on how to use it with maven or ant and checkout the example projects.

Current javadocs and a source xref is also available.

Download

You can download the current project sources in either zip or tar formats. The jars are available at maven central.

You can also clone the project with Git by running: 

$ git clone git://github.com/tcurdt/jdeb

get the source code on GitHub : tcurdt/jdeb
